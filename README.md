This repository contains the machine learning and diffraction image analysis pipeline developed to study the structural dynamics of viral capsids during dehydration.  

The project utilizes unsupervised machine learning to analyze X-ray single-particle imaging (SPI) data, allowing for the classification of diffraction patterns and the mapping of the structural landscape of MS2 bacteriophages as they transition from a hydrated to a dehydrated state.

This repository provides the computational pipeline used to analyze the structural landscape of MS2 bacteriophages under dehydration.
